## Usage
```bash
# Install
npm i -g vitals-scraper

# Run
vitals-scraper
```

### Options
- `--format` (default: `csv`)
- `--outFile` (default: `./mps.[format]`)
- `--parallel` (default: `5`)
- `--verbose` (default: `false`)
